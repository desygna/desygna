A system that enables `shadow` props.

| Prop         | CSS Property |
| ------------ | ------------ |
| `textShadow` | textShadow   |
| `boxShadow`  | boxShadow    |

```jsx
<Box textShadow="0 0 2px 3px #000" boxShadow="0 0 5px 10px red" />
```
